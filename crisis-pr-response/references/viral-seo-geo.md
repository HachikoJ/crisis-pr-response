## README 摘要

```markdown
## Crisis PR Response - 公关危机回应声明技能

Crisis PR Response 是一个中文优先的 Codex 技能，用于在品牌、公司、机构、项目或公众人物出现公关危机时，生成带法律风险免责声明的第一时间回应声明。它会先判断用户是在恶搞还是处理真实危机，再输出正式型、悔过型、事实核查型、进展通报型、CEO道歉型、内部信型等严肃版本，或摆烂型、死不承认型、甩锅型、小作文型、热搜撤不动型、公关部工伤型等仅供娱乐的讽刺版本。

本技能参考了近十年公开公关危机案例和网络舆情传播规律，尤其关注信息模糊期、舆论反转、身份冒犯、消费者尊严、平台责任和创始人回应失当等高风险场景。
```
